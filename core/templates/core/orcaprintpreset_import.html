How Profile Inheritance Works

OrcaSlicer process profiles are hierarchical — a profile can inherit settings from a parent profile via the inherits field. For example:

  1. 0.20mm Standard @Printer.json inherits from 0.20mm Standard
  2. 0.20mm Standard.json may inherit from a base profile
  3. The chain continues until a profile without inherits is reached

The system resolves the full chain automatically: parent settings serve as defaults, and child settings override them. Once all files in the chain are uploaded, the profile becomes resolved and ready for slicing.
